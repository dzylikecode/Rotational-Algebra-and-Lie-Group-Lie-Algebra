# 直线几何

## 基本概念

- 直纹面

  由直线运动构成的平面(任何一点都有直线穿过)

  方程:

  $$s(u, v) = a(u) + vl(u)$$

  与直线的参数方程同构:

  $$Line = \vec a + t * \vec p$$

  因而可类比出:

  $l(u)$ 代表了方向: 姿态向量

  $a(u)$ 代表了位置: 位置向量

  $v$ 相当于 t, 遍历一段区间

  > 关键是把直线的方向和位置分开来思考
